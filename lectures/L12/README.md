# L12 - Konvolutionella neurala nätverk (del I)

## Dagordning
* Introduktion till konvolutionella neurala nätverk (CNN) för bildklassificering.
* Fortsatt arbete med **P03**.

## Mål med lektionen
* Känna till varför vanliga neurala nätverk inte lämpar sig väl för bildklassificering.
* Känna till övergripande hur konvolutionella lager är uppbyggda.
* Kunna extrahera attribut ur bilder via kernels.

## Instruktioner
* Läs [bilaga A](./appendix/a_ann_for_images.md) för information om varför vanliga neurala nätverk
inte lämpar sig väl för bildklassificering.
* Läs [bilaga B](./appendix/b_cnn.md) för information om konvolutionella neurala nätverk.

## Utvärdering
* Varför lämpar sig vanliga neurala nätverk (enbart dense-lager) sämre för bildklassificering än ett CNN?
* Kan ni, i egna ord, beskriva syftet med respektive lagertyp i ett CNN: conv-, pooling-, flatten- samt dense-lager?
