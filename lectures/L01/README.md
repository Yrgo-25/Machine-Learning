# L01 - Linjär regression (del I)

## Dagordning
* Introduktion: Genomgång av kursens innehåll samt upplägg.
* Kort introduktion till `std::vector` (används genomgående i kursen).
* Vad är maskininlärning?
* Linjär regression: Prediktion, parametrar, fel, epoker och lärhastighet.
* Om tid finns, påbörja implementering av linjära regressionsmodellen i mjukvara.

---

## Mål med lektionen
* Känna till hur `std::vector` används i C++.
* Känna till varför ett traditionellt program inte lämpar sig väl för att efterlikna mänsklig intelligens.
* Känna till hur maskininlärning skiljer sig från traditionell programmering sett till indata, utdata och regler.
* Kunna träna enkla linjära regressionsmodeller för hand.
