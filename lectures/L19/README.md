# L19 - Förberedelse inför övningstentamen

## Dagordning
* Genomgång av centrala begrepp och exempeluppgifter inför övningstentamen.
* Möjlighet att ställa frågor om sådant som känns oklart.

## Mål med lektionen
* Känna sig förberedd inför övningstentamen i L20.
* Ha fått svar på eventuella frågetecken kring kursens innehåll.

## Förutsättningar
* Förståelse för konvolutionella neurala nätverk (**L12-L16**).

## Instruktioner
* Repetera gärna [föreläsningsmaterialet](../README.md) för L12-L16 samt tidigare lektioners bilagor.
* Ställ frågor om sådant som känns oklart inför övningstentamen.

## Utvärdering
* Känner ni er redo att genomföra övningstentamen på egen hand?
* Finns det några kvarstående frågetecken kring kursens innehåll?

## Nästa lektion
* Övningstentamen och kursavslut.
