## Dagordning

* Fortsatt arbete med **P03**.

## Mål med lektionen
* Kunna beräkna precisionen för en tränad regressionsmodell.
* Kunna avgöra när modellen är tillräckligt tränad utifrån ett givet tröskelvärde.
* Fortsätta arbetet med **P03**.
